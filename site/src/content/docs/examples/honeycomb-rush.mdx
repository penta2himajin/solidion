## About

A tower defense game ("Hanikamu Rush") showcasing Solidion's RECS at scale. Defend the honeycomb nest from waves of invading bugs using worker bees, while keeping the larvae in their "hanakamu" (blushing) state.

### RECS Validation Points

- **createIndex**: 4 reactive index sets (walking/blocked/slowed/dying)
- **System phases**: All three phases used — pre (unblock, slow field), main (movement, spawning, animation), post (block check, hit detection, goal check)
- **100+ entities**: Wave 4 reaches 100+ simultaneous enemies + projectiles
- **Hybrid architecture**: Enemies & projectiles via RECS, worker bees via hooks, larvae via Signals
